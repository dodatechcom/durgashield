About

DurgaShield v1.0.0

All-in-one privacy & security for Firefox: blocks ads, malware, phishing, crypto miners, and trackers. Includes anti-fingerprinting, social media isolation, element zapper, and more.

Privacy • Donate • GitHub • Report Issue • License

All Features

  • Ad Blocker — Blocks ads, banners, and video ads using filter lists
  • Malware Protection — Blocks known malicious domains and malware hosts
  • Crypto Miner Blocking — Prevents in-browser cryptocurrency mining scripts
  • Phishing Protection — Blocks fake login pages and phishing sites
  • Popup Blocker — Blocks unwanted popups and new window triggers
  • CDN Tracker Block — Blocks known CDN tracking domains
  • CDN Replacement — Serves local copies of CDN libraries to prevent tracking
  • Stealth Mode — Anti-fingerprinting, WebRTC block, removes URL tracking params
  • Never-Consent — Auto-dismisses cookie consent banners and GDPR notices
  • Enhanced Anti-Track — Spoofs browser fingerprints and tracking IDs
  • XSS Protection — Real-time DOM scanning to block cross-site scripting
  • ClearClick — Detects and neutralizes invisible clickjacking overlays
  • ABE (Anti-Beacon) — Local network protection against DNS beacons
  • Secure Payment — Blocks payment forms on insecure HTTP pages
  • Download Scanner — Scans downloads for malware and malicious files
  • Social Container — Isolates Facebook, Instagram, WhatsApp in separate containers
  • Social Media Filter — Blocks Facebook, Twitter, LinkedIn tracking widgets
  • Annoyance Blocker — Blocks cookie notices, push notifications, newsletter popups
  • YouTube Channel Whitelist — Allows ads on supported YouTube channels
  • Metadata Cleanup — Strips EXIF data from uploaded JPEG images
  • Search Annotations — Annotates search results with safety icons
  • Adult Content Block — Blocks adult and pornographic websites
  • Gambling Block — Blocks online gambling and betting sites
  • Violence & Hate Block — Blocks violent and hate speech content
  • Video Redirect Guard — Prevents hidden links in video players from opening tabs
  • HTTPS Enforcement — Upgrades HTTP connections to HTTPS on major sites
  • Password Leak Check — Auto-checks passwords against known breaches (HIBP)
  • Filter Log CSV Export — Downloads filter log as CSV file
  • Notification Batching — Queues and batches notifications to reduce noise
  • Tracker Map Limits — Automatically limits tracker map size
  • JavaScript Blocking — Block JS on specific site or globally (Advanced)
  • Browser Cleanup — Clear cache, cookies, history on schedule
  • Custom Rules — Add custom block/allow rules for any domain
  • Filter Lists — Manage 8+ filter lists (EasyList, EasyPrivacy, URLhaus, etc.)
  • Import/Export — Backup and restore all settings
